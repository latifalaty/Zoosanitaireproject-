ZoosanitaireProject

ZoosanitaireProject is a Python project that extracts animal-related information from URLs, filters it, and exports it to Excel. It uses AI models to process text in both French and English and features a Flask web interface for easy interaction.

Key Highlights

Tech Stack: Python, AI, Web Scraping, Flask

Functionality: Extract data from URLs → Filter relevant information → Export to Excel

AI Integration: Processes multilingual text to identify and structure animal-related data

Demo: included to showcase the workflow
